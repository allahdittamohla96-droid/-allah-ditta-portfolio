# Allah Ditta Mohla — Performance Marketing Portfolio

Portfolio website for Allah Ditta Mohla, Performance Marketing Specialist targeting the Dubai market.

## How to Deploy on GitHub Pages

1. Create a new GitHub repository named: `allah-ditta-portfolio` (or any name)
2. Upload ALL files from this folder to the repository root:
   - `index.html`
   - `profile.jpg`
3. Go to **Settings → Pages**
4. Under **Source**, select **Deploy from a branch**
5. Select branch: `main` and folder: `/ (root)`
6. Click **Save**
7. Your site will be live at: `https://YOUR-USERNAME.github.io/allah-ditta-portfolio/`

## Files
- `index.html` — Main portfolio page (all CSS and JS inline)
- `profile.jpg` — Profile photo
- `README.md` — This file

## Features
- Fully responsive (mobile + desktop)
- Custom gold cursor
- Scroll-reveal animations
- Working contact buttons (email, phone, LinkedIn)
- All 8 Dubai industry verticals showcased
- 9 key campaign results displayed
